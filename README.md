# Bank Management System (C++)

A simple **Bank Management System** built using **C++ with Object-Oriented Programing and File Handling**.
This Project allows users to manage bank accounts with persistent storage.

## Features

- Create Account
- Deposit Money
- Withdraw Money
- Display Single Account
- Display All Accounts
- Delete Account
- Persistent Storage using File Handling (`accounts.txt`)

## Technologies Used

- C++
- OOP (Classes, Objects, Encapsulation)
- STL (`vector`)
- File Handling (`fstream`)
- Multi-file Project Structure
